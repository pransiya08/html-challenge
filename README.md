# html-challenge
This project is deisgned to scrape data from the Mars News and Mars Weather Data websites using HTML.
The first part of this project uses HTML to scrape the article titles and previews from the Mars News website with all the scraped data saved into a Python dictionary.
The second part of this project uses HTML to scrape the information about Mars from the data table in the Mars Weather Data website. The scraped information are turned into a dataframe, which is outputed in a CSV file.
Data analysis and visualisations are also done using the Matplotlib functions.
